# Node Postit   

A simple application to save your postits!

## Dependencies

NodeJs or Docker

## Basic Usage

install dependencies

```
yarn install
```
config your .env based on .env.sample and run the command

```
yarn start
```

or run on a docker container with bash script!

```
bash make.sh install
```

see other options

```
bash make.sh help
```
